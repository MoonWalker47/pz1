# Мікросервісна інфраструктура логування (FastAPI + PostgreSQL)

Цей проект є практичною роботою з розгортання контейнеризованого веб-додатку, що взаємодіє з реляційною базою даних в ізольованому середовищі Docker.

## Структура стеку та сервісів
* **web (FastAPI):** Веб-додаток на Python, який надає REST API для запису та читання логів.
* **db (PostgreSQL):** База даних для надійного та постійного зберігання інформації.
* **pgadmin (pgAdmin 4):** Веб-інтерфейс для зручного адміністрування та перегляду таблиць БД.

## Конфігурація інфраструктури
* **Мережа (Network):** Усі контейнери об'єднані в ізольовану bridge-мережу `my_project_app_net`. База даних закрита всередині мережі і доступна тільки для додатку.
* **Томи (Volumes):** Для контейнера бази даних налаштовано Docker Volume `my_project_pg_data`. Це гарантує, що при перезапуску або видаленні контейнерів усі логи залишаються збереженими на жорсткому диску.

1. Переконайтеся, що у вас встановлено Docker та Docker Compose.
2. Склонуйте репозиторій або скопіюйте файли проекту в окрему директорію.
3. Запустіть стек у фоновому режимі:
```bash
sudo docker compose up -d --build
